# 截图目录

把下列截图放到本目录，主 README 的「截图展示」会自动显示。

| 文件名 | 对应页面 | 建议内容 |
|--------|----------|----------|
| `home.png` | 首页控制台 | 登录后首页：英雄区 + 统计卡片 + 最近动态 |
| `resume-edit.png` | 简历编辑 | 左表单 + 右实时预览 + 排版面板展开 |
| `files.png` | 文件管理 | 文件列表 + 文件夹 + 拖拽区 |
| `code.png` | 代码片段 | 代码高亮 + 运行结果 |
| `learn.png` | AI 学院 | 教程页（带目录侧栏） |
| `light-theme.png` | 亮色主题 | 任意页面的亮色模式 |

## 规格建议

- 格式：PNG
- 宽度：1280px（或 2x 高清图 2560px）
- 暗色主题截图为主，亮色单独一张对比

## 快速截图技巧

- Windows：`Win + Shift + S` 区域截图
- 浏览器整页截图：F12 → `Ctrl+Shift+P` → 输入 "screenshot" → Capture full size screenshot
